# 🧪 Proyecto Semana 03 - Calculadora de Dominio

## 📌 Descripción

Este proyecto consiste en desarrollar una **calculadora aplicada al dominio Sistema de Gestión de Laboratorio**, utilizando JavaScript y ejecutándose en **Node.js**.

El objetivo es aplicar los operadores aprendidos durante la semana para realizar cálculos y validaciones relacionadas con el funcionamiento de un laboratorio.

## 🎯 Dominio del Proyecto

**Sistema de Gestión de Labora**

# 🧪 Proyecto Semana 02 - Ficha de Datos del Dominio

## 📌 Descripción

Este proyecto consiste en crear una **ficha de datos en consola usando JavaScript**, aplicando los conceptos aprendidos en la semana 02 del bootcamp.

El script se ejecuta en **Node.js** y muestra información organizada sobre una entidad dentro del dominio **Sistema de Gestión de Laboratorio**.

Durante el desarrollo se aplican conceptos como:

* Uso de **variables con `const`**
* Tipos de datos primitivos
* Uso de **`typeof`**
* Conversión de tipos (`String()`, `Number()`, `Boolean()`)
* Uso de valores **null**
* Buenas prácticas de **nomenclatura de variables**

---

## 🎯 Dominio del Proyecto

**Sistema de Gestión de Laboratorio**

La ficha de datos representa un **equipo de laboratorio**, mostrando información relevante como:

* Nombre del equipo
* Cantidad disponible
* Estado de disponibilidad
* Técnico asignado
* Tipos de datos de cada variable

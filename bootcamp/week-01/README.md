# Sistema de Gestión de Laboratorio

Sistema web diseñado para administrar un laboratorio clínico. Permite gestionar pacientes, exámenes y resultados de manera organizada.

## 📋 Funcionalidades

- Registro de pacientes
- Gestión de exámenes
- Registro de resultados
- Control de inventario
- Reportes

## 🛠️ Tecnologías utilizadas

- HTML
- CSS
- JavaScript
- PHP
- MySQL

## 🚀 Instalación

1. Clonar el repositorio:

git clone https://github.com/calderonpenajuansebastian-rgb/sistema-gestion-laboratorio.git

2. Copiar los archivos a la carpeta `htdocs` (si usas XAMPP).

3. Importar la base de datos en phpMyAdmin.

4. Abrir en el navegador:

http://localhost/sistema-gestion-laboratorio

# 🧪 Proyecto - Sistema de Gestión de Laboratorio

## 📌 Descripción

Este proyecto consiste en crear un script en **JavaScript** que se ejecuta en **Node.js** y muestra en la consola una ficha de presentación de una entidad dentro del dominio **Sistema de Gestión de Laboratorio**.

El objetivo es aplicar los conceptos aprendidos en la semana 1:

* Uso de `console.log()`
* Tipos de datos primitivos (string, number, boolean)
* Comentarios en el código
* Expresiones aritméticas
* Organización de la salida en consola

---

## 🎯 Dominio del Proyecto

**Sistema de Gestión de Laboratorio**

La entidad principal presentada en el script es un **equipo de laboratorio**, mostrando información relevante como:

* Nombre del equipo
* Código identificador
* Categoría
* Cantidad disponible
* Precio del equipo
* Estados del equipo

---

## 🛠️ Tecnologías Utilizadas

* JavaScript ES2023
* Node.js

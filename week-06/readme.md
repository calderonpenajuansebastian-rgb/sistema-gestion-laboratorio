# Proyecto Semana 06 — Reporte con Bucles

## 📌 Dominio del Proyecto

**sistema-gestion-laboratorio**

---

## 🎯 Objetivo

Desarrollar un sistema de reporte en JavaScript que procese una colección de datos utilizando diferentes tipos de bucles.

El sistema analiza un conjunto de muestras de laboratorio, generando estadísticas, conteos y reportes detallados.

---

## ⚙️ Tecnologías utilizadas

* JavaScript
* Node.js

## 🚀 Cómo ejecutar el proyecto

1. Abrir la terminal
2. Ubicarse en la carpeta del proyecto
3. Ejecutar:

node starter/script.js

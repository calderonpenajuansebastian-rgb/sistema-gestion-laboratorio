# Proyecto Semana 05 — Clasificador

## 📌 Dominio del Proyecto

**sistema-gestion-laboratorio**

---

## 🎯 Objetivo

Desarrollar un sistema de clasificación en JavaScript aplicando estructuras de control y operadores modernos como:

* `if/else`
* operador ternario
* `switch`
* nullish coalescing (`??`)
* optional chaining (`?.`)

El sistema clasifica elementos del dominio de laboratorio y muestra una ficha informativa en consola.

---

## ⚙️ Tecnologías utilizadas

* JavaScript
* Node.js

## 🧩 Funcionalidades implementadas

El sistema realiza las siguientes operaciones:

### 1️⃣ Clasificación con if/else

Clasifica la capacidad del elemento en:

* Alta
* Media
* Baja

---

### 2️⃣ Estado con operador ternario

Determina si el sistema está:

* Disponible
* No disponible

---

### 3️⃣ Clasificación con switch

Clasifica el tipo de muestra:

* Biológica
* Química
* Física

---

### 4️⃣ Valor por defecto con ??

Asigna un responsable por defecto cuando no existe.

---

### 5️⃣ Acceso seguro con ?.

Permite acceder a propiedades de objetos sin generar errores si no existen.

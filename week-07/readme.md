# 🧪 Sistema de Gestión de Laboratorio

## 📌 Descripción
Este proyecto consiste en una librería de funciones reutilizables para gestionar equipos e insumos de un laboratorio.

## ⚙️ Funcionalidades
- Formateo de datos de equipos
- Cálculo de valor total
- Validación de disponibilidad
- Aplicación de descuentos
- Generación de reportes usando callbacks

## ▶️ Ejecución

1. Abrir terminal
2. Ejecutar:

node script.js

## 📊 Resultado
Se genera un reporte con:
- Información de cada equipo
- Estado (disponible o no)
- Valor total
- Precio con descuento
